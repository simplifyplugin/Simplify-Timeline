== Description ==
Simplify Timeline is WordPress plugin that provides a timeline.When user have history data then have to display data and date for that particular time.Its easy to use and configure.

There is many user want to display past time data.This plugin is on kind of supportive tool to make task easier.If user is not technical person can manage it well.

We are provide shortcode for display purpose.It enable you to move your timeline whenever and whereever.You have to add data into menu called SF Timeline.Add new and add data over there.

Use shortcode `sm_timeline`
